I embarked on the journey of developing a simple calculator using ReactJS as a fresher developer. This project posed several challenges that contributed to my learning experience.

Handling Input and Refs:

Validating and handling user input while using React refs for DOM manipulation were initial challenges. I had to ensure that only numeric values were accepted and correctly use useRef for input and result elements.

State Management:

Managing and updating state in a React component was a crucial aspect of the calculator. I used React useState to track and update the calculated result dynamically.

Preventing Default Form Submission:

Preventing the default form submission behavior to avoid page reloads was an important consideration in the button click functions. I used e.preventDefault() to handle this.

Resetting Input and Result:

Implementing functions to reset the input field and the calculated result required careful handling. I created resetInput and resetResult functions to manage these resets.

Error Handling for Mathematical Operations:

I implemented error handling to address scenarios like division by zero. This involved adding conditional checks in mathematical operation functions.

Testing:

Testing the calculator app to ensure it worked correctly and handled edge cases was important. I implemented unit tests using testing libraries like Jest and React Testing Library.
